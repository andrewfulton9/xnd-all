============
Installation
============

------------
From Source
------------

Clone the `xnd-all repo`_, cd into the repo and install.

.. code-block::

    $ git clone https://github.com/plures/xnd-all.git
    $ cd xnd-all
    $ python3 setup.py install







_`xnd-all repo` https://github.com/plures/xnd-all.git
