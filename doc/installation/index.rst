============
Installation
============

From Source
-----------

Clone the source for all xnd related libraries from the `Github repo`_.

.. code-block:: console

    $ git clone git@github.com:plures/xnd-all.git

From inside xnd-all repo:

.. code-block:: console

    $ ./configure
    $ make install
    $ make check
    $ python setup.py install
